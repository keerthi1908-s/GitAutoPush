# GitAutoPush

GitAutoPush is a simple Windows-based automation tool that automatically commits and pushes changes from your local repository to GitHub. It helps save time by handling repetitive Git operations with a single script.

## Features
- Automatically stages all changes in your local repository.
- Prompts for a commit message.
- Pushes changes directly to the main branch.
- Easy to use and beginner-friendly.

## Files
- `auto_git.bat` – The main batch script that automates Git add, commit, and push.
- `test.txt` – Sample text file used to test the automation script.

## Usage
1. Make sure Git is installed and configured on your system.
2. Open the project folder.
3. Double-click `auto_git.bat` or run it via Command Prompt.
4. Enter your commit message when prompted.
5. The script will automatically commit and push your changes.
